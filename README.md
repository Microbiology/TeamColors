# sportcolors
Color palettes from popular sports teams.

## How To Install
Type the following commands into your R session.

```r
library(devtools)
install_github("Microbiology/sportcolors")
library(sportcolors)
```

## Running an Example
Once it is installed, try it out.

```r
team_colors("Chicago Cubs")
```
